adjust module documentation¶

rotate_pt_by(x, y, deltatang, lx, ly)¶
rotate x,y about lx,ly adjusting for tilt given by deltatang
rotator(tang, xl, yl)¶

Given the difference in rotation between a ballot template and a particular ballot and the position of the landmark in the ballot template, return a function that given a coordinate pair in a template returns the corresponding coordinates in that particular ballot.

>>> r = rotator(0, 88, 122)
>>> r(98, 1030)
(98, 1030)
>>> r(464, 280)
(464, 280)
>>> r(832, 1746)
(832, 1746)
>>> r = rotator(.0687, 88, 122)
>>> r(98, 1030)
(160, 1027)
>>> r(464, 280)
(474, 254)
>>> r(832, 1746)
(942, 1691)
