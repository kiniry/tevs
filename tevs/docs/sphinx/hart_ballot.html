hart_ballot module documentation¶

class HartBallot(images, extensions)¶

Class representing ballots from Hart Intersystems.

Each Hart ballot has dark rectangles around voting areas, and voting areas are grouped in boxed contests.

build_layout(page)¶
get layout and ocr information
extract_VOP(page, rotate, scale, choice)¶
Extract a single oval, or writein box, from the specified ballot
find_landmarks(page)¶

retrieve landmarks for Hart images, set tang, xref, yref

Landmarks for the Hart Ballot will be the ulc, urc, lrc, llc (x,y) pairs marking the four corners of the main surrounding box.
